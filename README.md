# Wiki Enricher

Herramienta que busca un tema en Wikipedia, enriquece 
el contenido con IA, lo traduce y genera un informe 
en formato PDF o TXT.

## Descripción

Este proyecto permite al usuario investigar cualquier 
tema desde la terminal. Extrae información de Wikipedia, 
la mejora con OpenAI, la traduce con DeepTranslate y 
guarda el resultado en un archivo .txt o .pdf.

## Tecnologías

- Python 3
- BeautifulSoup
- Requests
- OpenAI API
- DeepTranslate
- ReportLab

Ejecuta el programa desde la terminal:

python main.py

El programa te pedirá:
1. Un tema para buscar en Wikipedia
2. El idioma al que traducir el contenido
3. El formato del archivo (.txt o .pdf)
4. El nombre del archivo
